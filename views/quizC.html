Q 4 - Does both the loops in the following programs prints the correct string length? 

            #include<stdio.h>
            
            main()
            {
               int i;
               char s[] = "hello";
            
               for(i=0; s[i]; ++i);
                  printf("%d ", i);
            
               i=0; 
               while(s[i++]);
                  printf("%d ", i);
            }

A - Yes, both the loops prints the correct length

B - Only for loop prints the correct length

C - Only while loop prints the correct length

D - Compile error in the program.

Answer : B

Explanation

In while loop ‘i’ gets incremented after checking for ‘\0’, hence giving 1 more than the length.

Q 9 - What value strcmp() function returns when two strings are the same?

A - 0

B - 2

C - 1

D - Error

Answer : A

Explanation

The C library function strcmp() compares two strings with each other and return the value accordingly.

               int strcmp(const char *str1, const char *str2)

Comparison happens between first string (str1) with second string (str2).

By comparing two strings, the values return by the function strcmp() are,

  • If, str1 is less than str2 then Return value < 0
  • If, str2 is less than str1 then Return value > 0
  • If, str1 is equal to str2 then Return value = 0
